# Usage #

- Run `berks_vendor.bash` to build the `build/cookbooks` directory
- Vagrant as usual

# Services In Box #
- redis: 6379 (default)
- postgresql: 5432 (default)
- mailcatcher: 1025 SMTP, 1080 HTTP (defaults)
- fakes3: 4567 (default for fakes3)
- fake_sqs: 4568
- nsqlookupd/nsqd: 4160 TCP, 4161 HTTP
- nsqadmin: 4171 HTTP
